# APBD Task 5

ASP.NET Core Web API for managing rooms and reservations in memory.

## Implemented features
- RoomsController with CRUD operations
- ReservationsController with CRUD operations
- route parameters, query string filters, and JSON body binding
- model validation with data annotations
- in-memory static datastore
- reservation overlap validation
- proper HTTP status codes

## Technologies
- ASP.NET Core Web API
- Controllers
- Swagger / Postman
- Static in-memory lists
